Subtle, purposeful animations reinforce the premium brand feel.

TokenValueUse
--wk-duration-fast150msHover states, focus rings
--wk-duration-base250msButtons, cards, toggles
--wk-duration-slow400msPage transitions, theme switch

The WeKruit site also uses a scroll-based entrance animation: large serif headlines fade in from a muted/transparent state as the user scrolls, creating a sense of depth and narrative flow.
